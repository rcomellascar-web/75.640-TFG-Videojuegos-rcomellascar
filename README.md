# 75.640-TFG-Videojuegos-rcomellascar

Instrucciones para poner en marcha el proyecto:

1. Tener instalado Node en la version 24.0.0 en adelante, si no se tuviera instalado se puede descargar desde el siguiente link: https://nodejs.org/es/download.
2. Copiar el repositorio haciendo ejecutando el comando de "git clone https://github.com/rcomellascar-web/75.640-TFG-Videojuegos-rcomellascar.git".
3. Abrir la carpeta en un editor de codigo como Visual Studio o WebStorm, hacer cd en la carpeta e instalar las dependencias con npm install.
4. Ejecutar con npm run dev.
